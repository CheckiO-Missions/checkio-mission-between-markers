Wie findet man den Text zwischen den Markierungen?
Durch die Verwendung eines Slice: 
        
def between_markers(text: str, begin: str, end: str) -> str:
    begin_index = text.find(begin) + len(begin)
    end_index = text.find(end)
    return text[begin_index:end_index]
begin_index - der Index des ersten Zeichens des Textes zwischen den Markierungen. end_index - der Index des ersten Zeichens der rechten Markierung. Da das Symbol an der end_index-Position nicht im Ergebnis enthalten ist, erhalten wir den benötigten Text. Mit anderen Worten, das letzte Zeichen, das tatsächlich Teil unserer Antwort ist, ist ein Zeichen an der Position 'end_index - 1', das das letzte Zeichen des Textes zwischen den Markierungen ist.
Was ist, wenn eine der Markierungen in der Zeichenkette fehlt?
Dann müssen Sie die Zählung an der Position 0 beginnen (wenn es keine linke Markierung gibt - begin) oder mit dem letzten Zeichen des Eingabetextes (wenn es keine rechte Markierung gibt - Ende). Falls die Eingabezeichenfolge keine Markierungen enthält, sollten Sie die gesamte Zeichenkette als Antwort zurückgeben.

def between_markers(text: str, begin: str, end: str) -> str:
    if begin in text:
        begin_index = text.find(begin) + len(begin)
    else:
        begin_index = 0
        
    if end in text:
        end_index = text.find(end)
    else:
        end_index = len(text)
        
    return text[begin_index:end_index]
Zunächst einmal prüfen wir, ob die linke Markierung im Text vorhanden ist. Wenn es keine gibt, beginnt die Antwort ganz am Anfang der Eingabezeichenkette. Wenn eine vorhanden ist, beginnt die Antwort mit dem Zeichen, das auf das letzte Zeichen der Markierung folgt. Danach suchen wir nach der richtigen Markierung in der Zeichenkette. Wenn es eine gibt, machen wir sie zur Begrenzung für unsere Eingabezeichenfolge. Wenn der Eingabetext die richtige Markierung nicht enthält, wird das Ende des Eingabetextes die Grenze unserer Antwort sein.

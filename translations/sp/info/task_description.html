Te dan una cadena y dos marcadores (el inicial y el final). Tienes que encontrar una subcadena encerrada entre estos dos marcadores. Pero hay algunas condiciones importantes:

  • Los marcadores iniciales y finales son siempre diferentes.
  • Si no hay un marcador inicial, entonces el primer carácter debe considerarse el comienzo de una cadena.
  • Si no hay un marcador final, entonces el último carácter debe considerarse el final de una cadena.
  • Si faltan los marcadores inicial y final, simplemente devuelve la cadena completa.
  • Si el marcador final viene antes que el marcador inicial, entonces devuelve una cadena vacía.

Entrada: Tres argumentos. Todos ellos son cadenas. Los argumentos segundo y tercero son los marcadores iniciales y finales.

SAlida: Una cadena.

Cómo se usa: para analizar textos

Condición inicial: no puede haber más de un marcador final y no puede haber más de una inicial

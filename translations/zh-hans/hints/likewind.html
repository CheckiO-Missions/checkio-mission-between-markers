def between_markers(text: str, begin: str, end: str) -> str:
    if begin in text:
        begin_index = text.find(begin) + len(begin)
    else:
        begin_index = 0
        
    if end in text:
        end_index = text.find(end)
    else:
        end_index = len(text)
        
    return text[begin_index:end_index]
首先，我们检查文本中的左标记。如果没有，那么答案将从输入字符串的开头开始。 开始。如果有，那么答案将从该标记最后一个字符之后的字符开始。 然后，我们检查字符串中是否有正确的标记。如果有，则将其作为输入字符串的极限。 如果输入文本不包含正确的标记，那么输入文本的末尾将是我们答案的极限。

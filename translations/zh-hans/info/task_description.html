给您一个字符串和两个标记（初始标记和最终标记）。您必须在这两个标记之间找到一个子串。但是有几个重要的条件：

  • 初始标记和最终标记总是不同的。
  • 如果没有初始标记，那么第一个字符应被视为字符串的开头。
  • 如果没有最终标记，则最后一个字符应视为字符串的结束。
  • 如果没有初始标记和最终标记，那么只需返回整个字符串。
  • 如果最终标记在初始标记之前，则返回空字符串。

输入 三个参数。所有参数均为字符串。第二个和第三个参数是初始标记和最终标记。

输出： 字符串。

举例说明： 

{{init_code_tmpl}}

如何使用： 用于解析文本

前提条件： 不能有多个最终标记和多个初始标记。标记不能为空字符串
